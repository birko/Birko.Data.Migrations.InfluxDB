# Birko.Data.Migrations.InfluxDB

InfluxDB migration provider for the Birko Framework.

## Features

- Bucket and retention policy management
- Data lifecycle migrations
- InfluxDBMigration base class

## Installation

```bash
dotnet add package Birko.Data.Migrations.InfluxDB
```

## Dependencies

- Birko.Data.Core (AbstractModel)
- Birko.Data.Stores (Settings)
- Birko.Data.Migrations
- Birko.Data.InfluxDB
